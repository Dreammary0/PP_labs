Что делает программа?
Есть одно машинное слово (W) и вектор из нескольких машинных слов (V). Нужно найти остаток от деления V на W.

Как она работает?
Вектор делится на части и для каждой части считается остаток, затем для этих частей тоже вычисляется остаток, и так в итоге приходим к результату.
Проблема с памяться для машинного слова решается кольцом Z.

Какие файлы за что отвечают?
* main.cpp: Управление тестами и вывод результатов.
* vector_mod.cpp: Основная логика вычислений.
* mod_ops.cpp: Математические операции по модулю.
* num_threads.cpp: Управление количеством потоков.
* performance.cpp: Измерение производительности.
* randomize.cpp: Генерация тестовых данных.
* test.cpp: Тестовые данные для проверки корректности.
* config.h: Общие определения и константы.
* CMakeLists.txt: Конфигурация сборки проекта.

-------------------
Программа начинается с выполнения функции main в файле main.cpp. Затем идет проверка на корректность работы функции vector_mod на тестовых данных из test.cpp.
Если все тесты проходят успешно, программа переходит к измерению производительности. Программа запускает функцию vector_mod с разным количеством потоков
(от 1 до максимального количества потоков на машине). Для каждого количества потоков измеряется время выполнения функции.
Результаты измерений записываются в файл output.csv и выводятся в консоль.
