Что делает программа?
Программа выполняет операции с большими массивами данных (векторами) с использованием модульной арифметики и многопоточности.

Как она работает?
Программа проверяет корректность работы функции vector_mod на тестовых данных, затем измеряет её производительность с разным количеством потоков.

Какие файлы за что отвечают?
main.cpp: Управление тестами и вывод результатов.
vector_mod.cpp: Основная логика вычислений.
mod_ops.cpp: Математические операции по модулю.
num_threads.cpp: Управление количеством потоков.
performance.cpp: Измерение производительности.
randomize.cpp: Генерация тестовых данных.
test.cpp: Тестовые данные для проверки корректности.
config.h: Общие определения и константы.
CMakeLists.txt: Конфигурация сборки проекта.

-------------------

                                    Как работает программа
Запуск программы: Программа начинается с выполнения функции main в файле main.cpp.

Проверка корректности: Программа проверяет корректность работы функции vector_mod на тестовых данных из test.cpp.
Если все тесты проходят успешно, программа переходит к измерению производительности.

Измерение производительности: Программа запускает функцию vector_mod с разным количеством потоков
(от 1 до максимального количества потоков на машине). Для каждого количества потоков измеряется время выполнения функции.

Запись результатов: Результаты измерений записываются в файл output.csv и выводятся в консоль.

Завершение работы: Программа завершает работу, выводя результаты тестов и производительности.
